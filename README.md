## Разработка Системы Управления Задачами - TaskManagementService

### Локальный запуск проекта

Запуск осуществляется с помощью [*docker-compose.yml*](docker-compose.yml) 
``` bash
docker-compose up -d
```

### Swagger
Swagger-ui доступен на  [*http://localhost:8080/swagger-ui/#/*](http://localhost:8080/swagger-ui/#/)
